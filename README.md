# Student Success Prediction

## Machine Learning Problem
The goal of this project is to predict whether a student will pass or fail based on academic and behavioral factors. This is a supervised machine learning classification problem.

## Dataset
The dataset contains information about students including study hours, attendance rate, previous GPA, and quiz scores. These features are used to predict the final outcome (Pass or Fail).

## Features
- Study Hours
- Attendance Percentage
- Previous GPA
- Number of Quizzes Taken

## Target Variable
- Final Result (Pass / Fail)
